في هذا اليوم نتعرف على استخدام مكتبة matplotlib
للتمثيل البياني للانحدار الخطي والقيم المتوقعة
تحتاج لتثبيت المكتبة باستخدام الأمر:
pip install matplotlib
وكذلك نستخدم مكتبة Scikit-learn 
لحساب الانحدار الخطي .
لتثبيتها نستخدم الامر :
pip install scikit-learn
